## Features:
`FoldColumnToggle` toggle fold column between 0 3

## test on

## Installation
To install the plugin copy `autoload`, `plugin`, `doc` directories into your `.vim` directory.
Vim plugin management is also support vim-plug, Pathogen, et. al.
